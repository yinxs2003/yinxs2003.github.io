给你一个整数数组 nums，请你将该数组升序排列。 

示例 1：

输入：nums = [5,2,3,1]

输出：[1,2,3,5]



示例 2：

输入：nums = [5,1,1,2,0,0]

输出：[0,0,1,1,2,5]



使用堆排序进行排序操作，堆排序可以分成两个步骤

1. 构建大顶堆（小顶堆）
2. 打印或者保存第一个节点
3. 交换第一个和最后一个节点，继续执行第一步

代码如下

```java
package com.code.note.test;

import com.code.note.tree.BinaryHeap;

import java.util.Arrays;

public class Solution2 {

    public void buildBigHeap(int[] arr, int parent, int len) {

        int child = parent * 2 + 1;
        while (child < len) {
            if (child + 1 < len && arr[child] < arr[child + 1]) {
                child++;
            }

            if (arr[parent] > arr[child]) break;

            // 交换子节点和父节点
            int t = arr[parent];
            arr[parent] = arr[child];
            arr[child] = t;

            parent = parent * 2 + 1;
            child = parent * 2 + 1;
        }

    }

    public void heapSort(int[] arr) {
        for (int i = arr.length; i >= 0; i--) {
            buildBigHeap(arr, i, arr.length);
        }

        BinaryHeap b = new BinaryHeap();
        b.insertLevelOrder(arr);
        b.print();

        for (int i = 0; i < arr.length; i++) {

            // 将大顶堆第一个元素放到数组末尾
            int tmp = arr[arr.length - i - 1];
            arr[arr.length - i - 1] = arr[0];
            arr[0] = tmp;

            //大顶堆
            buildBigHeap(arr, 0, arr.length - i - 1);
        }

    }

    public static void main(String[] args) {

        Solution2 s = new Solution2();

        int[] arr = new int[]{1, 3, 41, 55, 2};


        s.heapSort(arr);

        BinaryHeap b2 = new BinaryHeap();
        b2.insertLevelOrder(arr);
        b2.print();


        System.out.println(Arrays.toString(arr));
    }
}
```
